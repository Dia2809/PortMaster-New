## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|Right stick|Camera|
|R1 + DPAD / Left stick|Camera (alternative)|
|A|Jump|
|B|Dash|
|X|Hook|
|Y|Save data|
|Select|Menu|
