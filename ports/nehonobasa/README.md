## Controls

| Button | Action |
|--|--| 
|dpad|movement |
|left analog|movement |
|B button|fire special weapon|
|A button|shoot/rapid fire|
|X button|fire special weapon|
|R2|shoot/rapid fire|
|L2|fire special weapon|
