## Controls

| Button | Action                 |
| ------ | ---------------------- |
| D-PAD  | Move Cursor            |
| A      | Rotate Clockwise       |
| B      | Rotate Counterclockwise|

## How To Install  

Buy HEXA on itch -> https://raewtf.itch.io/hexa  
Download the Universal zip file, unzip it and place 'HEXA.love' in ports/hexa

Enjoy!
