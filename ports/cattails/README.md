## Controls

| Button | Action |
|--|--| 
|Left stick|Movement|
|Right stick|Use skills|
|A|Interact|
|B|Cancel|
|Y|Skills|
|X|Inventory/Map|
|R1|Item|
|R2|Attack|
|L2|Sneak|
