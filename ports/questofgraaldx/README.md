ROCKNIX users must use the Panfrost (not libMali) graphics driver.


## Controls

| Button | Action   |
| ------ | -------- |
| D-PAD  | Movement |
| B/UP   | Jump     |
| A/L1   | Attack   |
| Y/L2   | Special  |
| START  | Start    |
| SELECT | Back     |
