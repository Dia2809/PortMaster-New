## Controls

| Button | Action |
|--|--| 
|DPAD / Left Stick|Movement|
|A|Confirm|
|B|Roll|
|X|Attack|
|Y|Change weapon|
|L1/R1|Scroll inventory|
|R|Special attack|
|Start|Inventory|
|Select|Menu|
