## Controls

| Button  | Action            |
| ------- | ----------------- |
| D-PAD   | Menu Navigation   |
| START/Y | Click menu items  |
| L-STICK | Sword rotation    |
| A       | Large Sword Swing |
| B       | Small Sword Swing |
| X       | Restart level     |
| SELECT  | Pause Menu        |
