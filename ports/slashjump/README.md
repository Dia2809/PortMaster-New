## Controls

| Button | Action |
|--|--| 
|DPad|Menu Navigation|
|L Analog|Sword Rotation|
|Select|Pause Menu/Restart Level|
|R1|Restart Level (MuOS)|
|A|Large Sword Swing|
|B|Small Sword Swing|
