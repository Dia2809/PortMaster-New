## Controls

Controls are customizable, but the typical layout is this:

| Button | Action |
|--|--| 
|DPAD / Left stick|Steering|
|A/B or R2/L2|Gas/Break|
|R1 / L1|Clutch|
|Start|Menu|
|Select|Restart|
