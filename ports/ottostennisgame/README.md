## Controls

| Button  | Action               |
| ------- | -------------------- |
| X       | Player 1 jump        |
| UP      | Player 2 jump        |
| D-PAD   | Navigate menus       |
| A/START | Activate menu item   |
| SELECT  | Pause / in-game menu |
