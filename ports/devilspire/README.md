## Controls

| Button | Action |
|--|--| 
|Left stick|Movement|
|Right stick|Aim|
|DPAD Left/Right|Item selection|
|DPAD Down|Use selected item slot|
|DPAD Up|Spell Scroll selection|
|A|Dodge|
|B|Interact|
|X|Kick|
|Y|Inventory|
|L1|Defend|
|R1|Attack|
|L2|Run|
|R2|Spell Cast|
|L3|Map|
|Start|Pause|
|Select|Lamp|
