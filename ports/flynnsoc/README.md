## Controls

| Button | Action |
|--|--| 
|D-pad / L-stick|Movement|
|A|Jump|
|B|Roll|
|X|Attack|
|Y|Heal|
|L1|Switch Magic|
|L2|Cast Magic|
|R1|Switch Weapon|
|R2|Crimson Control|
|Start|Menu|
|Select|Inventory|
