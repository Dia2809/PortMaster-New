## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick|Movement|
|A|Shoot / Confirm|
|B|Jump|
|L1|Boost|
|R1|Special|
|Start|Pause menu|
|Select|Options menu \ Leave options menu|
